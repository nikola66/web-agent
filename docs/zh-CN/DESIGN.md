# Web Agent 设计系统

Web Agent 是以复古未来主义终端为核心的浏览器原生 AI 工作区。视觉语言应更接近严肃编码代理，而非泛 SaaS 仪表盘：直接、紧凑、发光、可操作。

## 设计原则

- 输出优先，而非 chrome 优先。终端与代理 transcript 是体验中心。
- 复古终端气质，现代执行。老式命令行线索 + 干净高对比 finish。
- 紧凑但可读。信息密度高而不拥挤。
- 发光强调色，深色基底。光少量用于焦点、状态与 affordance。
- 浏览器原生、本地优先。UI 应像用户机器上的工具，而非伪装成桌面的远程网页。

## 视觉识别

受 Claude Code、Codex、OpenCode 及 OpenClaw、Hermes Agent 终端 UX 启发。应感觉：工程感、即时、略带戏剧感、技术可信。

终端区应像现代 TUI：等宽输出、锐利对比、极少装饰、清晰命令反馈。其他表面更安静，让终端保持视觉锚点。

## 色彩系统

以黑为主、品红紫能量。

### 核心表面

- `#000000` - 主背景
- `#0a0a0a` - 表面背景
- `#111111` - 抬升表面

### 文字

- `#ffffff` - 主文字
- `#bcbcbc` - 次要文字
- `#666666` - 弱化文字

### 品牌强调

- `#4b1cdd` - violet
- `#6823e5` - purple
- `#8a38f5` - magenta dark
- `#c633f7` - magenta
- `#fb75fc` - magenta light
- `#be32d6` - CTA accent

### 光晕与边框

- 光晕柔和，非霓虹过重。
- 强强调 `rgba(251, 117, 252, 0.5)`，环境光 `rgba(251, 117, 252, 0.14)`。
- 边框细而克制：白色 8–18% 不透明度。

### 语义色

- Success: `#34d399`
- Warning: `#fbbf24`
- Error: `#f87171`

## 字体

主字体：`Poppins`

- UI chrome、标签、导航用 `Poppins`
- 终端输出、代码、命令与技术表面应等宽、工具感

规则：用字号与字重对比而非过多样式；标签小而明确；终端区避免消费级 SaaS 文案风格。

## 几何

- `8px` - 小控件
- `13px` - 按钮
- `34px` - 卡片
- `54px` - 面板
- `64px` - 板
- `124px` - 药丸形

## 动效

像设备而非装饰：`150ms` / `200ms` / `250ms`，`cubic-bezier(0.16, 1, 0.3, 1)`。用于启动、悬停、加载/思考、profile 与工作区切换。避免无目的持续动画。

## 终端语言

签名表面。语气：能干的代理控制台；错误与状态行清晰紧凑。xterm 主题贴近核心色板（黑底、白字、品红光标、半透明品红选区）。

## 组件与布局

分层代理工作区：终端与 transcript 为垂直锚点；侧栏密集可用。移动端保持相同层级。

## 品牌文案

界面文案：直接、技术、负载下冷静、自信不浮夸。避免 filler。

## 避免

- 泛 pastel SaaS
- 把终端压成聊天气泡 UI
- 过度渐变/光晕
- 单屏混太多视觉系统
- 辅助面板压过代理运行时

## 实现

真源：`src/styles.css`、`src/ui/theme.ts`。变更时先改 token，再对齐组件。
