# 贡献者公约行为准则

## 我们的承诺

我们作为成员、贡献者与维护者，承诺让每个人都能在无骚扰环境中参与，不论年龄、体型、可见或不可见残疾、族裔、性别特征、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教或性取向。

我们承诺以有助于开放、欢迎、多元、包容与健康社区的方式行动与互动。

## 我们的标准

积极行为示例：对他人展现同理与善意；尊重不同观点；给予并优雅接受建设性反馈；为错误承担责任并道歉、学习；关注社区整体利益。

不可接受行为示例：性化语言或意象、性骚扰；挑衅、侮辱或人身/政治攻击；公开或私下骚扰；未经明确许可发布他人隐私信息；在专业场合下明显不当的其他行为。

## 执行责任

社区领导者负责澄清与执行标准，对不当、威胁、冒犯或有害行为采取适当公正措施，并可在适当时说明 moderation 原因。

## 适用范围

适用于所有社区空间，以及个人在公共场合代表社区时（官方邮箱、社媒、指派代表等）。

## 执行

可通过 [SUPPORT.md](SUPPORT.md) 所列渠道向维护者报告。所有投诉将及时公正审查。领导者须尊重报告者隐私与安全。

## 执行指南

### 1. 纠正

**影响**：不当语言或其他不专业行为。**后果**：私下书面警告并说明违规性质。

### 2. 警告

**影响**：单次或系列违规。**后果**：警告及继续违规的后果；在指定期限内避免与当事人及执行者非必要互动。

### 3. 临时封禁

**影响**：严重违反社区标准。**后果**：指定期限内禁止与社区公开或私下互动。

### 4. 永久封禁

**影响**：持续违规、骚扰个人或针对群体的攻击。**后果**：永久禁止在社区内公开互动。

## 归属

改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1。社区影响指南受 [Mozilla 执行阶梯](https://github.com/mozilla/diversity) 启发。
